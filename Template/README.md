# Templates

Reusable HTML templates and shared visual assets for the v-ofernando workspace.

## Files

| File | Purpose |
|------|---------|
| `Workshop Template.html` | AI Agents Workshop deck — visual reference (cover, agenda, section-head, cards, refs-row, JS interactions). Consumed by [`agent/copilot-readiness.agent.md`](../agent/copilot-readiness.agent.md) to build `copilot-chat-readiness.html`. |
| `Icons/`, `other.icons/`, `partner.icons/` | Local icon assets used by `Workshop Template.html` |
| `Screen.prints/` | Screenshots used during template iteration |

> **Note:** The M365 Roadmap template and its agents were moved to the [`roadmap/`](../roadmap/) folder.
