Glossary — A to Z

~60 terms commonly tested on AI-900. Each entry: 1–2 line definition + link to the official Microsoft Learn page where the term is canonically defined.

Jump: A B C D E F G I K L M N O P R S T U V

A

Accountability
Responsible AI principle: people who design and deploy AI systems remain responsible for outcomes. ref
Accuracy
Classification metric: (TP + TN) / total predictions.
Anomaly detection
AI workload that flags unusual / outlier data points. ref
AutoML
Azure ML feature that automatically tries algorithms + hyperparameters to find the best model for your data. ref
Azure AI Foundry
Unified portal (formerly Azure AI Studio) to build, evaluate, and deploy generative AI apps. URL: ai.azure.com. ref
Azure AI Language
Unified service for sentiment, key phrases, NER, PII, summarization, conversational language understanding. ref
Azure AI Search
Search service used as the knowledge mining + RAG retrieval layer (formerly Cognitive Search). ref
Azure AI Speech
Speech-to-text, text-to-speech, speech translation, speaker recognition. ref
Azure AI Vision
Image analysis (tags, captions, objects, OCR, faces). ref
Azure Machine Learning
End-to-end ML platform: notebooks, AutoML, designer, pipelines, registered models, endpoints. ref
Azure OpenAI Service
Microsoft-hosted access to OpenAI foundation models (GPT-4o, DALL·E, Whisper, embeddings) with enterprise security. ref

B

Bing chat answers
Consumer-facing generative AI search, powered by Azure OpenAI. (Not testable in depth on AI-900.)

C

Classification
Supervised ML task predicting a category (binary or multi-class).
Clustering
Unsupervised ML task grouping similar items without labels.
Computer vision
AI field for understanding images and video. Azure service: Azure AI Vision.
Confusion matrix
Table of TP / FP / FN / TN used to evaluate a classifier.
Content filter
Azure OpenAI feature that moderates prompts and completions for hate, sexual, violence, self-harm. ref
Conversational Language Understanding (CLU)
Azure AI Language feature for intent + entity detection (replaces LUIS). ref
Copilot (Microsoft 365)
Generative AI assistant integrated into Word, Excel, PowerPoint, Outlook, Teams.

D

DALL·E
OpenAI image-generation model, available via Azure OpenAI.
Designer (Azure ML)
Drag-and-drop pipeline builder for training/inference in Azure ML Studio.
Document Intelligence
Service for extracting text, tables, key-value pairs from forms (formerly Form Recognizer). ref

E

Embedding
Numeric vector representation of text/image, used for semantic similarity and retrieval (RAG).
Endpoint
HTTP URL exposed by an Azure AI service or deployed model; called with a key or managed identity.
Entity recognition (NER)
Identifying people, places, organizations, dates in text. Azure AI Language feature.

F

F1 score
Harmonic mean of precision and recall — preferred over accuracy for imbalanced classes.
Face service
Detects, identifies, verifies faces. Part of Azure AI Vision. ref
Fairness
Responsible AI principle — model behaves equitably across demographic groups.
Feature
Input variable used by a model to make predictions.
Fine-tuning
Continuing training of a base LLM on your own labeled data to specialize behavior. ref
Foundation model
Large pre-trained model adaptable to many downstream tasks (GPT-4o, Llama, Phi).

G

Generative AI
AI that creates new content (text, image, audio, code).
GPT-4o
OpenAI multimodal foundation model (text + vision + audio).
Grounding
Connecting LLM responses to authoritative data (RAG) to reduce hallucination.

I

Inclusiveness
Responsible AI principle — AI must serve people of all abilities and backgrounds.
Inference
Running a trained model to make predictions on new data.

K

Key phrase extraction
Azure AI Language feature returning the main topics of a text.
Knowledge mining
AI workload that surfaces insights from unstructured content using search + AI enrichment.

L

Label
Target value in supervised learning (the answer the model learns to predict).
Large Language Model (LLM)
Foundation model trained on text; GPT-4o is an LLM.

M

MAE (Mean Absolute Error)
Regression metric — average absolute residual.
Microsoft 365 Copilot
Generative AI in M365 apps; uses Microsoft Graph for grounding.
Multimodal
Model that handles multiple input types (text + image + audio).

N

Named Entity Recognition
See Entity recognition.
Natural Language Processing (NLP)
AI field for understanding human language. Azure services: Language, Speech, Translator.

O

OCR (Optical Character Recognition)
Reading printed/handwritten text from images. Azure AI Vision feature.
Overfitting
Model fits training data too closely; fails to generalize.

P

PII detection
Azure AI Language feature for finding/redacting Personally Identifiable Information.
Precision
TP / (TP + FP) — proportion of positive predictions that are correct.
Privacy & Security
Responsible AI principle — protect user data, respect consent.
Prompt
Input text/instructions sent to an LLM.
Prompt engineering
Crafting prompts (zero-shot, few-shot, system message) to guide LLM output. ref
Prompt flow
Azure AI Foundry feature for orchestrating LLM apps with chaining, evaluation, deployment.

R

RAG (Retrieval-Augmented Generation)
Pattern: retrieve relevant docs → pass as LLM context → grounded response.
Recall
TP / (TP + FN) — proportion of actual positives caught.
Regression
Supervised ML task predicting a continuous numeric value.
Reliability & Safety
Responsible AI principle — system performs as expected, handles unanticipated conditions safely.
Responsible AI
Microsoft framework of 6 principles for ethical AI. ref
RMSE
Root Mean Square Error — regression metric, penalizes large errors more than MAE.

S

Semantic kernel
Microsoft open-source SDK for building LLM apps with skills, planners, memory.
Sentiment analysis
Azure AI Language feature returning positive / neutral / negative with confidence scores.
Speech-to-text (STT)
Azure AI Speech feature transcribing audio to text.
Supervised learning
Training on labeled data (features + known target).
System message
Initial instruction in chat completion defining model role/behavior.

T

Text-to-speech (TTS)
Azure AI Speech feature synthesizing audio from text.
Token
Unit of text (~4 chars EN) used for LLM billing and context-window limits.
Transparency
Responsible AI principle — users understand how the system makes decisions.
Translator
Azure AI service for text translation between 100+ languages.

U

Unsupervised learning
Finding patterns in data without labels (clustering, anomaly detection).

V

Validation
Subset of training data held out to tune hyperparameters / select models.
Video Indexer
Azure service extracting faces, keywords, topics, sentiment from video. ref
