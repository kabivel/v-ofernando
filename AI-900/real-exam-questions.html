⚠️ Outdated terminology you'll meet in these questions (still asked in dumps; product names changed):
  • QnA Maker → retired 31 Mar 2025. Replacement: Azure AI Language → Custom Question Answering
  • LUIS (Language Understanding) → retired 1 Oct 2025. Replacement: Azure AI Language → Conversational Language Understanding (CLU)
  • Form Recognizer → renamed to Azure AI Document Intelligence (2023)
  • Computer Vision API / Computer Vision service → renamed to Azure AI Vision (May 2023)
  • Text Analytics → absorbed into Azure AI Language
  • Cognitive Services → renamed to Azure AI Services (2023)
  • Azure AI Studio → renamed to Azure AI Foundry (Nov 2024)
  • Azure ML Designer "modules" → called "components" in v2 SDK/CLI (the v1 Designer UI still works, but maintenance mode)
  • Personalizer / Metrics Advisor / Anomaly Detector → announced for retirement on 1 Oct 2026
  • Face service identification / verification → Limited Access since Jun 2022 (requires Microsoft approval). Emotion / age / gender detection were retired in 2022.
  • AKS deploy for ML endpoints → superseded by Managed Online Endpoints (MOE) in AML v2. Azure Container Instances deploys are being deprecated.
  • "Cognitive Services" multi-service resource → renamed to "Azure AI services" multi-service resource
  • Computer Vision v3 'Categories' / 'Landmark' / 'Celebrity' domains → legacy v3 API; new development uses Azure AI Vision Image Analysis 4.0. 'Celebrity' domain was removed for Responsible AI reasons (2020).
  • AI-900 exam retires 30 Jun 2026 — replaced by AI-901

Golden rules from this set:
  • Predict a numeric value → Regression
  • Predict a category → Classification
  • Your own images → Custom Vision; ready-made tasks → Computer Vision
  • Create new features → Feature engineering; reduce/select → Feature selection
  • Bounding boxes = Object detection; label + % only = Image classification
