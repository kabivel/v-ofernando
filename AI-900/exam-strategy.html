Exam-day strategy

3-pass technique

PassTime per questionAction
Pass 1 — Fast30–45 sAnswer easy ones immediately. Flag anything that needs >45s and move on.
Pass 2 — Medium60–90 sTackle flagged. Use elimination heuristics from pegadinhas.html.
Pass 3 — Reviewremaining timeRe-read still-flagged. Never leave blank — eliminate 2 and guess (25% → 50%).

Time budget (assuming 50 questions in 60 min)

  • Pass 1: 35 min (target: answer 35–40 questions)
  • Pass 2: 20 min (target: clear ~10 flagged)
  • Pass 3: 5 min (review what's left, sanity-check unsure ones)

When to flag

  • Two answers look equally plausible
  • Service name you've never heard of (probably old name like "QnA Maker")
  • Question has a long scenario you can't parse in 20 s
  • You're tempted to second-guess — flag, move on, return with fresh eyes

Question-type tactics

TypeTactic
Single-choiceRead all 4 options before deciding. Often "obvious right" answer is a trap; check option 4.
Multi-select ("choose 2")Answer count is given — eliminate 2 wrong ones first.
Drag-and-dropMap service → use case. Use process of elimination if unsure of one mapping.
Yes/No seriesEach yes/no is independent — read each carefully; don't assume a pattern.
Scenario / case studyRead the question before the scenario. Then scan scenario for the specific keyword you need.

Night before

  • Light review of cheatsheet.html + mistake-log.html only. No new material.
  • Confirm exam scheduled time, ID, and proctoring environment.
  • If online proctored: tidy your workspace, no second monitor, charger plugged in, water bottle clear/label removed.
  • Sleep ≥ 7 hours.

Morning of

  • Eat normally. Avoid heavy caffeine if not your usual.
  • Skim cheatsheet.html ONCE — Responsible AI principles + service → workload mapping.
  • Arrive / log in 15 min early.
  • Bathroom break before starting (timer doesn't pause).

During exam

  • You can't return to previous questions in Microsoft fundamentals exams ONLY IF case-study sections. Verify on the intro screen.
  • If allowed: flag liberally on first pass.
  • Read every option before clicking — Microsoft loves "best answer" among multiple defensible ones.
  • Watch for negatives: "Which is NOT…", "Which is the LEAST…".
  • If completely lost: identify what the question is about (which domain), eliminate options that belong to other domains.

After submitting

  • Result is immediate. Score / 1000, pass at 700.
  • Pass: download badge from Credly within 48h via Learn profile.
  • Fail: free retake offered (depends on promo). Use mistake-log.html to focus.
