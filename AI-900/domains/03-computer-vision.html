Domain 3 — Computer vision on Azure

Weight: 15–20% · Skill focus: match Azure services to vision tasks; understand image-related capabilities.

Sections: Capabilities · Services · Decision tree · Renames · Self-check

1. Computer-vision capabilities

CapabilityWhat you give itWhat you get back
Image classificationImageOne (or top-N) label(s) for whole image
Object detectionImageList of objects + bounding boxes + labels
Image captioning / descriptionImageNatural-language sentence
OCR (Read API)Image / PDFPrinted + handwritten text, line/word coordinates
Face detectionImageBounding box per face, basic attributes
Face identification / verificationImage + person groupPerson ID / match confidence (Limited Access)
Form / receipt / invoice extractionForm image / PDFKey-value pairs, tables, structure
Brand / landmark detectionImageKnown brands / landmarks detected

2. Azure services for vision

ServicePrimary usePre-trained?Custom?
Azure AI VisionGeneral image analysis + OCRYesLimited (Image Analysis 4.0 customization)
Azure AI FaceFace detection / identification / verificationYesPerson groups
Azure AI Custom VisionTrain your own image classifier / object detectorNo (you bring data)Yes
Azure AI Document IntelligenceForms, invoices, receipts, IDsPre-built modelsYes (custom models)
Azure AI Video IndexerVideo — extract faces, keywords, sentiment, topicsYesPerson/brand models

3. Decision tree — which service?

  • "I need to caption / tag / OCR generic images" → Azure AI Vision
  • "I need to identify a specific person" → Azure AI Face (Limited Access)
  • "I want to train a model on my own labeled images" → Azure AI Custom Vision
  • "I'm extracting fields from a form / invoice / receipt" → Azure AI Document Intelligence
  • "I'm processing video" → Azure AI Video Indexer

5. Self-check

  1. You receive 1000 photos and want a list of objects in each. Service + capability?
  2. You want to extract the supplier name and total from PDF invoices. Service?
  3. You want to know who is in a security camera photo. Service?
  4. You only need to know how many faces are in a photo. Service is enough?
  5. You want to train a model to recognize your company's products. Service?

Answers: 1. Azure AI Vision — object detection · 2. Azure AI Document Intelligence (pre-built invoice model) · 3. Azure AI Face (Limited Access required) · 4. Azure AI Vision (basic face detection — bounding boxes) · 5. Azure AI Custom Vision
