3. Azure AI Translator

  • Text translation between 100+ languages (REST API + SDK)
  • Document translation (preserve formatting in Word/PDF)
  • Custom Translator: train domain-specific models
  • Audio translation belongs to AI Speech, NOT Translator — common exam trap

Reference: Translator overview

4. Decision tree — which NLP service?

  • "I need sentiment / key phrases / NER / PII / summarization" → Azure AI Language
  • "I'm building a chatbot's brain (intent + entity recognition)" → Azure AI Language — CLU
  • "I want a Q&A knowledge base over FAQs / docs" → Azure AI Language — Question answering
  • "I need to transcribe audio / synthesize speech" → Azure AI Speech
  • "I need to translate spoken language live" → Azure AI Speech — Speech translation
  • "I need to translate written text only" → Azure AI Translator

5. Self-check

  1. Customer reviews — you need overall sentiment per review. Service + feature?
  2. You want a chatbot that understands "book me a flight to Lisbon tomorrow". Service + feature?
  3. You want a help-desk bot answering FAQs from your internal KB. Service + feature?
  4. You need to transcribe a 1-hour meeting recording. Service?
  5. You need to translate a contract from English to Portuguese. Service?

Answers: 1. Azure AI Language — sentiment analysis · 2. Azure AI Language — CLU (intent + entities) · 3. Azure AI Language — Question answering · 4. Azure AI Speech (batch transcription) · 5. Azure AI Translator (document translation)
