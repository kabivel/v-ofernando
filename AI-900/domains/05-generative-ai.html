1. What is generative AI?

Generative AI creates new content (not just predictions). Built on foundation models — large pre-trained models adaptable to many tasks.

2. Azure OpenAI Service

Microsoft-hosted access to OpenAI models with enterprise controls.

CapabilityDetail
Models availableGPT-4o / GPT-4 family, GPT-3.5, DALL·E 3, Whisper, text-embedding-3
AuthAPI key or Entra ID (preferred — managed identity)
NetworkPublic endpoint + Private Endpoint option
Content filterOn by default — hate / sexual / violence / self-harm (prompts + completions)
Region availabilityLimited Azure regions per model
Data privacyYour prompts/completions not used to train OpenAI models
AccessApplication form required for some models (responsible AI review)

vs OpenAI direct: same models, plus enterprise security, regional hosting, RBAC, SLA, private networking, default content filters.

5. RAG vs Fine-tuning — pick the right tool

RAGFine-tuning
What changesPrompt context (retrieved docs)Model weights
Best forUp-to-date / private knowledge, citationsFormat consistency, style, narrow task
Setup costLow — index your data + queryHigh — labeled examples + training
Latency / cost per callHigher (extra retrieval call)Standard
Update frequencyReal-time (re-index)Re-train
Hallucination riskLower (grounded)Same as base unless data covers cases
Azure componentsAzure AI Search + Foundry + OpenAIAzure OpenAI fine-tuning

Rule of thumb: start with RAG; fine-tune only when you need consistent formatting or behavior.

7. Responsible generative AI

Microsoft uses a 4-stage model: Identify → Measure → Mitigate → Operate.

  • Identify potential harms (hate, manipulation, IP leak)
  • Measure harm frequency and severity (eval datasets)
  • Mitigate at 4 layers: model · safety system · metaprompt + grounding · UX
  • Operate with monitoring, incident response, transparency

Reference: Responsible AI for Azure OpenAI

8. Self-check

  1. You want your chatbot to answer using your company's policy documents. RAG or fine-tuning?
  2. What is the unit of LLM billing and context-window measurement?
  3. Which Azure portal hosts the model catalog and prompt flow?
  4. You want consistent JSON output formatting from an LLM — RAG or fine-tuning?
  5. Is the Azure OpenAI content filter enabled by default? Can it be removed easily?
  6. What grounds Microsoft 365 Copilot's answers?

Answers: 1. RAG · 2. Token (~4 chars EN) · 3. Azure AI Foundry portal (ai.azure.com) · 4. Fine-tuning · 5. Yes — on by default for prompts + completions; removal requires an application/approval · 6. Microsoft Graph (your tenant's emails, files, chats)
