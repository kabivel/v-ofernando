Mock exam schedule

Only two official sources are used:

  1. Microsoft official free practice assessment — 50 questions, unlimited attempts, identical UI to real exam.
  2. simulado.html in this kit — 50 questions, written from official Microsoft Learn modules.

No third-party "exam dumps", paid mock services, or community question banks are referenced. Using them violates Microsoft's exam policies and risks certification revocation.

Mandatory after each mock

  1. Add every wrong question to mistake-log.html (use the "📝 Add to mistake-log" button in simulado.html).
  2. For each wrong answer, read the linked official doc. Don't just memorize the answer.
  3. If you got it wrong because the service was renamed, update your pegadinhas awareness.
  4. If 3+ wrong in the same domain → re-read that domain page before the next mock.

Stopping criteria — schedule the real exam when:

  • Two consecutive practice runs ≥ 80%
  • All Responsible AI principle questions answered correctly
  • No "renamed service" traps surprise you anymore
  • You finish the 50 questions in < 50 min on at least one attempt
