Top 20 exam traps + 10 elimination heuristics

Each trap: seems-true → actually-true → why.

Traps

1. "Inclusiveness means using diverse training data."
→ Inclusiveness = AI serves people of all abilities and backgrounds (accessibility focus). Diverse data relates to Fairness.
2. "If a model explains its decisions, that's Accountability."
→ That's Transparency. Accountability = humans remain responsible for AI outcomes (governance).
3. "Form Recognizer is the right service for invoices."
→ Form Recognizer was renamed to Azure AI Document Intelligence. Pick the new name.
4. "Cognitive Search" is on the exam.
→ Renamed Azure AI Search. Both names may still appear; both refer to the same service.
5. "LUIS is the right service for chatbot intent detection."
→ LUIS is retired (deprecated). Use Conversational Language Understanding (CLU) in Azure AI Language.
6. "QnA Maker"
→ Replaced by Question answering feature in Azure AI Language.
7. "Azure AI Studio"
→ Renamed Azure AI Foundry portal (ai.azure.com).
8. Confusing accuracy vs F1 for imbalanced classes.
→ Accuracy is misleading when one class dominates (e.g., 99% non-fraud). Use F1 or precision/recall.
9. "Use accuracy to evaluate a regression model."
→ Accuracy is a classification metric. Regression uses RMSE / MAE / R².
10. "Clustering needs labeled data."
→ Clustering is unsupervised — no labels required.
11. "Designer in Azure ML requires Python code."
→ Designer is drag-and-drop, no-code. Notebooks are code-first.
12. "Computer Vision can identify specific people."
→ Generic Computer Vision returns face bounding boxes only. Identification/verification needs the Face API (Limited Access — requires approval).
13. "OCR and image classification are the same."
→ OCR extracts text. Image classification assigns a label to the whole image.
14. "Object detection returns one label per image."
→ Object detection returns multiple objects with bounding boxes; image classification returns one (or top-N) label(s) for the whole image.
15. "Speech translation = Translator service."
→ Azure AI Speech handles speech translation (audio in, transcript out in target language). Translator is text-only.
16. "Azure OpenAI and OpenAI direct are identical."
→ Azure OpenAI adds enterprise security, regional hosting, RBAC, private networking, SLA, content filters. Same models, different governance.
17. "RAG = fine-tuning."
→ Not the same. RAG retrieves docs at runtime to ground responses. Fine-tuning permanently adjusts model weights via training.
18. "A higher temperature makes the model more accurate."
→ Higher temperature = more creative / random. For factual answers, use low temperature (0–0.3).
19. "Content filter in Azure OpenAI is optional."
→ Content filter is on by default for both prompts and completions. You can request modifications but cannot trivially disable it.
20. "Microsoft 365 Copilot uses public ChatGPT data."
→ M365 Copilot is grounded on Microsoft Graph (your org's emails, files, chats) with enterprise data boundary — not public ChatGPT.

10 elimination heuristics

  1. If the question mentions "unlabeled data" → answer is clustering / unsupervised / anomaly detection. Never regression/classification.
  2. If the question is about extracting text from a document → OCR (general images) or Document Intelligence (forms/invoices).
  3. If the question mentions specific person identification → Face API (not generic Computer Vision).
  4. If the question mentions chatbot, intent, utterance → CLU (Conversational Language Understanding).
  5. If two answers describe the same Responsible AI principle in different words, look for keyword anchors: accessibility=Inclusiveness, explain=Transparency, responsible humans=Accountability.
  6. If the question asks about creative writing or image generation → generative AI / Azure OpenAI.
  7. If the question asks about grounding or company data with an LLM → RAG (not fine-tuning).
  8. If a service name sounds old (Form Recognizer, LUIS, QnA Maker, Cognitive Search, AI Studio) — check whether it was renamed. Modern names are usually the right answer.
  9. If the question asks for a no-code ML solution → Azure ML Designer or AutoML (not Notebooks).
  10. Words like "always", "never", "only" in an answer are usually wrong on AI-900.
