D2 · Machine learning (14 cards)

Q: Supervised learning — definition. D2
A: Training on labeled examples (features + known target)
Q: Unsupervised learning — definition. D2
A: Finding patterns in unlabeled data — clustering, anomaly detection
Q: ML task predicting a continuous number? D2
A: Regression
Q: ML task predicting a category? D2
A: Classification
Q: ML task grouping similar items without labels? D2
A: Clustering
Q: Typical train/validation/test split? D2
A: 70 / 15 / 15 (or 80/10/10)
Q: Regression error metric in original units? D2
A: RMSE (or MAE)
Q: Formula: TP / (TP+FP)? D2
A: Precision
Q: Formula: TP / (TP+FN)? D2
A: Recall
Q: What is F1? D2
A: Harmonic mean of precision and recall
Q: What is a confusion matrix? D2
A: Table of TP/FP/FN/TN for classification evaluation
Q: What is overfitting? D2
A: Model memorizes training data, fails on new data
Q: Azure service for no-code AutoML + designer + endpoints? D2
A: Azure Machine Learning (ml.azure.com)
Q: Which Azure ML feature is drag-and-drop? D2
A: Designer

D5 · Generative AI ⭐ (12 cards)

Q: Service hosting GPT-4o, DALL·E, Whisper? D5
A: Azure OpenAI Service
Q: Central portal for building generative AI apps? D5
A: Azure AI Foundry (ai.azure.com) — formerly Azure AI Studio
Q: What is a token? D5
A: Unit of text (~4 chars EN); billing + context-window unit
Q: What is prompt engineering? D5
A: Crafting input prompts to guide LLM toward desired output
Q: What is RAG? D5
A: Retrieval-Augmented Generation — retrieve docs then pass as LLM context to ground responses
Q: Azure OpenAI vs OpenAI direct? D5
A: Azure adds enterprise security, regional hosting, RBAC, private networking, SLA, content filters
Q: What is Microsoft 365 Copilot? D5
A: Generative AI in Word/Excel/PPT/Outlook/Teams, grounded on Microsoft Graph
Q: What is a system message? D5
A: Initial instruction defining LLM persona/behavior in chat completion
Q: What is grounding? D5
A: Connecting model responses to real data sources (e.g., RAG) to reduce hallucination
Q: What does the content filter do? D5
A: Moderates prompts and completions for hate/sexual/violence/self-harm (on by default)
Q: What is fine-tuning? D5
A: Continuing training of a base model on your labeled data to specialize behavior
Q: Generative AI workloads besides text? D5
A: Image generation (DALL·E), code (Copilot/Codex), audio (Whisper for STT)
